# 🎓 UIU Course Code Decoder (what-is-this-course)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Chrome Web Store](https://img.shields.io/badge/Chrome_Web_Store-v1.0.0-green.svg)](#)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](CONTRIBUTING.md)

Never guess a course code again! This Chrome Extension automatically decodes UIU (United International University) course codes (e.g., `CSE 3421`) into their full descriptive names (e.g., `Database Management Systems`) on any webpage.
